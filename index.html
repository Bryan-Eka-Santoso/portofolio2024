Pengalaman Pramuka

Saya mengikuti pramuka mulai dari saat saya SD, saya menjadi tim aktif di sekolah SD saya pada tahun 2017 - 2019. Saya juga menjadi perwakilan di sekolah saya dalam mengikuti Lomba Tingkat 2 di Sasana Krida Jati Jejer.

Lalu saya melanjutkan pengalaman organisasi Pramuka saya saat saya SMP pada tahun 2019 - 2020. Pada tahun 2019 di sekolah SMP saya mengadakan Perkemahan Jumat Sabtu. Setelah saya mengikuti Perkemahan Jumat Sabtu, saya terpilih menjadi Tim Inti Pramuka di sekolah SMP saya.

Banyak sekali pengalaman dan kebersamaan yang sudah saya lalui bersama teman teman saya saat saya mengikuti organisasi pramuka, seperti perkemahan, pelantikan, dan sebagainya.

Pengalaman OSIS

Awal saya mengikuti OSIS pada saat saya SMP, pada tahun 2019 - 2020. Saya juga terpilih menjadi Ketua OSIS sehingga saya harus memimpin OSIS dengan baik. Banyak sekali pengalaman yang saya dapatkan saat saya menjadi seorang Ketua OSIS. Seperti memimpin sebuah rapat, penanggung jawab sebuah acara, menjadi contoh dan teladan pada seluruh siswa, dan lain sebagainya.

Lalu saya memilih untuk melanjutkan pengalaman OSIS saya pada saat saya SMK pada tahun 2021 - 2022. Pengalaman yang saya dapatkan sangatlah banyak, karena pada saat saya SMK lebih banyak sekali acara yang di adakan di banding pada saat saya menjadi OSIS saat SMP. Saya mempelajari banyak sekali hal, seperti contohnya saya harus beradaptasi pada lingkungan baru, berkomunikasi dengan teman baru, dan lain sebagainya.

Pengalaman PKL

PKL di sekolah saya menjadi salah satu syarat untuk kelulusan. Kelas saya di berikan waktu selama 9 bulan untuk melaksanakan PKL. Saya PKL di PT. Multi Manao Indonesia, sebagai IT. Saya PKL bersama 3 teman saya, dan di berikan tugas untuk membuat sebuah aplikasi peminjaman barang. Saya di berikan tugas untuk menjadi Front End Developer dan Database Engineer.

Tugas dari Front End Developer yaitu, merancang ide dasar tampilan website atau aplikasi. Tugas dari Database Engineer yaitu, merancang, memelihara, dan mengoptimalkan infrastruktur data untuk pengumpulan, pengelolaan, transformasi, dan akses data.

Tidak hanya fokus pada sebuah aplikasi peminjaman barang saja, saya dan teman teman saya juga di berikan tugas mengentry sebuah data di Ms. Excel dan Spreadsheet, memperbaiki sebuah kabel jaringan, merakit kembali sebuah PC, memperbaiki PC, dan lain sebagainya.
